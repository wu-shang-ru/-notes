先到要fork的網站 點選fork按鈕
之後到自己本身的git去複製網址
然後創建一個資料夾 輸入 git clone 網址

(由於git一定要有資料才可以做確認，所以這邊我們先新增一個記事本裡面內容隨意)
之後在有.git的資料夾 去做git status 先做資料確認
確定有一個剛剛的資料 然後做 git add .

git commit -m "xxx"

git push -u origin master
如果是其他分支就把master提換成相應的名稱

再次檢查狀態 git status

然後做git remote add upstream https://github.com/Frizo-studio/Frizo.git
(增加一個遠端上游 名稱取為 upstream 後面的網址是fork的clone)

git remote -v   做遠端的檢查
會看到本地端的origin 及 剛剛新增的上游 upstream 

git pull upstream mastar
可以將上游的資料更新至本地端

git push -u origin master
將本地端資料上傳上去

記得最後都要發PR完成一個循環
